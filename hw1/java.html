Java was created by James Gosling, Mike Sheridan, and Patrick Naughton in June of 1991. It was originally supposed to be a for interactive television but it was too advanced for its time. Originally named Oak after an Oak tree outside of James Gosling's office, then renamed to Green, and finally renamed Java. Currently Java is one of the most popular computer languages and it is used for Android programming. It has 5 goals of java:

Simple, object-oriented, and familiar

Robust and secure.

Architecture-neutral and portable

Execute with "high performance.

Interpreted, threaded, and dynamic.
