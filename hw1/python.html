Python was created in 1989 by Guido van Rossum in the Netherlands and it first released in 1991. The language was developed to emphasize code readability, and syntaxt that allows less lines of code. Python is an open source software.

Python's core philosophy are:

Beautiful is better than ugly

Explicit is better than implicit

Simple is better than complex

Complex is better than complicated

Readability counts
